# EnqueueZero Techshack 2019-07

大家好，这是 EnqueueZero Techshack 2019-07 期。以下内容存档于 2019-07 CN (opens new window); 其英文版笔记存档于2019-07 (opens new window)。

Kubernetes 社区有人在考虑用 Rust 写一份 Kubernetes 的实现，cloudatomiclab.com (opens new window)。有人写了一个 kubectl-fzf 的工具可以改进 kubectl 的使用体验，做到模糊查找，github.com (opens new window)。CVE-2019-5736 是新近爆出的在 runc 上的漏洞，建议所有运行 docker 的地方都升级新版本。 在 Kubernetes Edge 使用 Ambassador 将配置桥接到 Envoy v2 上，工程挑战不小,kubernetes.io (opens new window)。Kubernetes 的配置管理有颇多方案，但没有一个是完美的,blog.argoproj.io (opens new window)。最后一篇关于 Kubernetes 的分享是欧洲核子研究中心（CERN）如何使用 Kubernetes。

数据库 Sharding 的原理：digitalocean.com (opens new window)。一个因为 systemd timeout 默认配置过短而导致集群卡住的用例：percona.org (opens new window)。

一些编程相关的文章有：好好锻炼写 SQL 的能力！craigkerstiens.com (opens new window)。好好学习写 GraphQL 的能力！hashnode.com (opens new window)。好好学习写 TypeScript 的能力！developer.okta.com (opens new window)。好好学习如何配置 Nginx！github.com (opens new window).

关于云计算的文章有：一篇来自 Berkeley 的关于 Serverless 现状的分析 rise.cs.berkeley.edu (opens new window)。为什么有时候 AWS Autoscale 没能 Scale？segment.com (opens new window)

一些关于工程实践相关的文章有：如何在 Amazon 写文档给 Boss 看，usejournal.com (opens new window)。如何在你的组织里面使用 RFC 流程来完成各种变更？philcalcado.com (opens new window)。GraphQL 的一些使用原则 principledgraphql.com (opens new window)。不要只追求测量代码覆盖率，应该去做 RCA，做监控，提交能覆盖 bugfix 的用例，jamesshore.com (opens new window)。尽可能使用结构化的日志事件，避免使用文本日志，charity.wtf (opens new window) 。

监控相关的文章：一篇老文章 - 来自 Google 工程师的自白 - 如何做监控告警 docs.google.com (opens new window)。Kapacitor 如何做 Scaping Discovery docs.influxdata.com (opens new window)。
